# Fatboy POS Frontend

Aplicacion web React + Vite + TypeScript + Tailwind.

Comandos desde la raiz del monorepo:

```bash
npm run dev:frontend
npm run build:frontend
npm run lint:frontend
```

Comandos desde `apps/frontend`:

```bash
npm run dev
npm run build
npm run lint
```

Este paquete debe mantenerse como UI/presentacion. La logica de negocio y el acceso a datos reales deben vivir en `apps/backend`.
